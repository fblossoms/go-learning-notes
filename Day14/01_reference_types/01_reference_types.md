# 引用类型

- 切片（复制header包括指针）
  - 根据原sliceheader拷贝一份相同的、新的sliceheader，原、新slicehaader（地址不同）里的ptr指向同一个底层数组
  - 减少大规模数据移动
  - 后续改变元素时，两切片有影响

- 数组
  - 根据原底层数组拷贝一份相同的、新的底层数组，两底层数组地址不同
  - 后续改变元素时，两数组互不影响

- 复制地址（指针）也叫做传址（本质是大整数）完全的副本，本质来说就是值复制

# Go语言中全部都是值传递（没有严格意义的引用类型），整型、数组这样类型的值是完全复制，slice、map、channel、interface、function这样的引用类型也是值拷贝，不过复制的是标头值
