# 子切片

- 从一个切片上截取一段叫做子切片
- 格式
  - 切片名[start（起始索引）:end（截止索引+1）] // start <= end，不写start和end相当于完全拷贝
  - 截取范围，前包后不包 [start, end-1]，start和end的最大可写值是cap
  - 缺省start时默认为0，缺省end时默认为len
  - len = end - start
  - cap = 从ptr指向的首地址往后算
    - 若ptr指向的首地址不变，结束索引变，则使用同一个底层数组（首地址一致，不产生新的底层数组）
    - 若ptr指向的首个地址改变，则使用不同数组（首地址不一致，产生新的底层数组）
