# 结构体
- Go中的结构体有点像面向对象语言中的“类”，但不完全是，Go也没打算真正实现面向对象范式
- 语法

```go
        type 名称 struct {
```

    - // n个字段的定义
    - 字段名 类型

```txt
        }
```

- 定义新类型
  - 实例化：用这个类型来可以定义n个该结构体的实例，把实例赋值给变量
  - u1.name // 属性
  - u1.getName // 方法（没加()表示不调用）
  - 把数据和方法合称为成员member，有数据成员和方法成员
- 有属性/字段
  - 数据有类型
- 方法method
  - 语法示例：func （u User) getName() string {}
  - 本质就是操作，也就是函数u1.getName，函数签名就是func (u User) string
