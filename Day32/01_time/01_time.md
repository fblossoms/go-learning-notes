# 时间
- 时间是非常重要的，离开了时间，生产环境数据将毫无意义

- 时区
  - UTC     世界协调时（发达国家之间的实验室的铯原子钟的时间，几千万年误差一秒），相当于零时区的GMT
  - GMT     格林威治时间
  - local   timezone时区，GMT+所在的时区（西时区为-）

- 格式化
  - 时间对象.Format(layout) -> string
  - 2006/01/02 15:04:05 -0700
  - 0 有几位就显示几位，哪怕是0
  - 9 有几位就显示几位，最后是无效数字，舍弃

- 解析
  - 格式化字符串和其他字符一致
  - 对于小数
    - 0 位置要对齐，长度和小数一致
    - 9 可以对应所有小数部分

- 时间戳（JavaScript常用毫秒时间戳）
  - 从1970-1-1 0时开始到现在所经历的秒数，同一时间世界上任意地区的时间戳一致

```txt
        GMT -> local

```

- 时区
  - 统一采用零时区或时间戳
  - 记录时间时，保存时区信息

- 时间构造

```txt
        time.Now()
        time.Date(...)
        time.Unix(sec, nesc)

```

- 时间运算
  - 时间 + 时间 没有意义
  - 时间 - 时间 -> 时间差delta
  - 时间 + 时间差 -> 新的时间对象
