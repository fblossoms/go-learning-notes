# 排序

- 对容器中的元素按照一定的原则排序
  - map如何排序
    - 先要进行线性化：非线性数据结构先要处理成线性数据结构，即遍历元素，把所有元素装到线性表里
    - 有了线性表后就可以进行排序了

- 线性数据结构
  - 切片slice、数组array

- 使用sort包进行排序，排序操作默认为升序（由小到大），排序为就地排序，调整同一个底层数组中元素的位置
  - sort包（排序后称为sorted）
    - Ints 快捷方法，对[]int进行排序，排序时切片header、len、cap都不变，下同
    - Strings 快捷方法，对[]string进行排序
    - 自定义类型以后再说，需要结构体、接口的知识

    - 降序
      - sort.Sort(sort.Reverse(强制类型转换为满足需要的接口类型))

- 排序算法
  - 效率O(n*n)
    - 交换排序
      - 冒泡法
        - 两两比较，谁大谁就放在右边
        - 交换次数过多，性能比选择排序慢
    - 选择排序
      - 每一趟选择一个极值放到一端，形成一个有序区，无序区减小
      - 下一趟对无序区选择极值，放到一端
    - 插入排序
      - 定一个哨兵值，选择合适插入点插入
      - 小规模效率还行
  - 效率O(n*log(n))：效率好
    - 归并排序
    - 快速排序（快排）
      - 递归函数调用，用交换的方式，小的元素放左边，大的元素放右边

  - 理想是O(n)，但是不可能的
  - 多数语言的在规模大时，使用快排；规模小时，采用插入排序
  - sort.Sort() 内部有多种排序算法，根据规模来使用不同算法

  - 总结
    - 排序都很慢，快排也很慢，规模越大越耗时
