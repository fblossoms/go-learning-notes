# MongoDB
- 属于非关系型数据库
- 内部使用类似于Json和bson二进制格式
- 文件
  - admin
    - 从权限的角度来看，这是一个“root”数据库
    - 要是将一个用户添加到这个数据库，这个用户自动继承所有数据库的权限
    - 一些特定的服务器端命令也只能从这个数据库运行，比如列出所有的数据库或关闭服务器
  - local
    - 这个数据永远不会被复制，可以用来存储限于本地单台服务器的任意集合
  - config
    - 当Mongo用于分片设置时，config数据库在内部使用，用于保存分片的相关信息
  - ┌──────────────┬──────────────────────────────┐
  - │ RDBMS        │ MongoDB                      │
  - ├──────────────┼──────────────────────────────┤
  - │ Database     │ Database                     │
  - │ Table        │ Collection                   │
  - │ Row          │ Document                     │
  - │ Column       │ Field                        │
  - │ Join         │ Embedded Document 或 Reference│
  - │ Primary Key  │ 主键（MongoDB 提供 _id）       │
  - └──────────────┴──────────────────────────────┘
  - 必须有主键
- MongoDB集合
  - 库
    - 集合collection 表table
    - 字段Field column
      - 相同字段，即名称和类型一致
    - 可以事先没有库、集合（身份认证、权限）、字段，在这个集合中，每一行（每一条文档）的字段可以完全不同。一般来说，不要把不同字段的文档放在同一个集合中
  - 文档
    - 文档应该有主键，默认_id 类型是ObjectID
    - 看似是Json的格式，底层是二进制存储的Bson（B为二进制的首字母）
  - 数据类型
    - 条件
    - string 2 有长度，包含结尾的null字符的1个字节
    - int 16
    - ObjectId 7
      - 不同驱动库生成的id策略略有不同
      - []byte 数组
        - [0:4]   4bytes  时间戳
        - [4:9]   5bytes  进程id
        - [9:12]  3bytes  基础数字（随机数），每次加1
- CRUD增删改查
  - 增加
    - Insert* 建议主键使用默认
      - 序列化 持久化 文件格式

  - 查询
    - 使用Bson构建查询条件，构建一个数据结构对象作为查询条件
    - 游标Cursor
      - 对返回的结果集逐行指向的针
      - 有的可以回头，有的不能回头（不能回头就装进切片容器，用切片的做法读取），一般不支持索引
    - 查询条件
      - 使用bson.D、bson.M
      - 查询效率
      - 简单条件
        - 推荐使用bson.M
      - 指令
        - $xxx 各有格式，各不相同
          - 比较
            - $in、$nin
          - 与或非
            - $and、$or、$not
    - 查询类型
      - bson.D（切片类型，不推荐，花括号嵌套太多，不利于阅读）
        - 类型为primitive.D

```txt
                    primitimv.D -> type D []E
```

        - bson.D{内容} -> primitive.D{} -> []E{} -> []有键值对内容的结构体类型
        - 内容元素
          - E{Key:"name", Value:20} E类型的实例，可以写n个元素
      - bson.M（推荐）
        - map，键值对
        - bson.M{内容}
        - 类型为primitive.M
      - bson.A（用的少）
        - 有序元素的数组
        - bson.A{x, y, z}
        - 类型为primitive.A
      - bson.E（用的少）
        - element 一个元素
      - 类型{内容}，字面量定义
        - []int{内容}
  - 更新
    - 使用指令
    - UpdateOne
      - UpdateByID
    - UpdateMany
    - 对匹配的文档进行一个或多个字段的值的更新
    - 替换Replace
  - 删除
    - 慎用
