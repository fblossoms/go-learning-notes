# 指针

地址，就像房间编号，本质是一个大整数

使用逻辑地址通过公式映射为物理地址

内存要编址，就像联排的宿舍依次连续的房间号码，从0开始按照字节编号，称为线性编制

从类型来说，专门定义了pointer，细分为*int（同样地，有*int8、int16等）、*string等（类型的值使用内存方式不同，数据结构不同）

- *int 指针类型，指向一个int类型值的地址（门牌号）

    ```go
    var a = 100     // a是int类型，是由关联值决定（100）的
    b := &a     // Go中只能在变量前写&用来取地址，b被赋值，b就有了类型，类型为*int
    ```

- *string 指针类型

## 举例：

```go
var a = 100
b := &a
fmt.Println(&a, b)
```
