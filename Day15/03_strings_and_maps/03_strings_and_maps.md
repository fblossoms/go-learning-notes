# 字符串

- 只读，内存不可变（内容、长度均不可变），不可修改
  - 字符相加，如"abcd" + "ef"，是开辟了一个新空间存放，并未改变原内存

- 值类型，和整数很像

- 双引号界定符
  - "" 空串，字面常量，长度为 0
  - "ab\t\nc" 支持转义字符

- Go语言中，字符串底层字节数组使用了UTF-8编码，每个汉字使用3字节，英文兼容ASCII，每个占用1字节

- 元数据StringHeader（分为ptr和len）
  - ptr 指针
  - len（int类型，8字节） 记录字符串字节序列长度，字节总数（Go语言取字节数）

- 可以使用索引和切片（子串，没有cap）

- 反引号界定符
  - `a\nb` ``里不支持转义字符 \n，不等于 "a\nb"，自行实现转义功能。适合实现直接粘贴文章、json格式等
  - 如果想写反引号怎么办？做不到。支持转义字符，但不支持 \n

- 拼接
  - + ：拼接
  - Join：使用间隔符拼接字符串切片
  - Builder：多次拼接，推荐
  - fmt.Sprintf：方便快捷，推荐

- 查询
  - Index：从左至右搜索，返回子串第一次出现的字节索引位置。未找到，返回 -1。子串为空，也返回 0。
  - LastIndex：从右至左搜索，返回子串第一次出现的字节索引位置。未找到，返回 -1。
  - IndexByte、IndexRune 与 Index 类似；LastIndexByte 与 LastIndex 类似。
  - IndexAny：从左至右搜索，找到给定的字符集字符串中任意一个字符就返回索引位置。未找到返回 -1。
  - Contains 方法本质上就是 Index 方法，只不过返回 bool 值，方便使用 bool 值时使用。
  - LastIndexAny 与 IndexAny 搜索方向相反。
  - Count：从左至右搜索子串，返回子串出现的次数。

- 时间复杂度是 O(n)，效率不高，该用则用，但要少用。
