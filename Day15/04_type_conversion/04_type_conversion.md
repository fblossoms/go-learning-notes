# 类型转换

- 数值类型转换（注意辨别场景，有些场景不可以类型转换，例如财务方面，这会丢失精度）
  - 低精度向高精度转换可以，高精度向低精度转换会损失精度
  - 无符号向有符号转换，最高位是符号位
  - byte和int可以互相转换
  - float和int可以相互转换，float到int会丢失精度
  - bool和int不能相互转换
  - 不同长度的int和float之间可以互相转换

- string(正整数) Unicode码点找字符，字符放在字符串里

- 定义新类型
  - 语法
    - type 新类型 旧类型
  - 作用
    - 定义枚举
      - 让你写的东西，在枚举定义的常量范围之内，如果不在，就给你报错
    - 扩展新方法
      - 更加灵活方便
