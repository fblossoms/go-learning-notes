# 接口
- 接口类型断言
  - 对接口而言，可以判断并断定变量标识符是什么具体的类型

```txt
            var a = 123 // 不是断言，因为编译器能推断出是int类型
            var b any = a // 是断言。b既是int类型，又是any类型
```

  - 断言有可能是错的
  - 断言的语法，只能用于接口变量
    - 变量 := 接口变量.(Type)
      - 断言是可以失败的，失败会报错panic
      - 变量：断言成功后的值
    - 变量, 布尔变量 := 接口变量.(Type)
      - 布尔变量为true说明断言成功
      - 布尔变量为false说明断言失败，有布尔值时不会报错
