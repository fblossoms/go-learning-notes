# 返回值
- 返回值也是值，定义时不返回。调用时分配在栈帧中保存该值
  - 是为当前函数调用的栈帧中产生的返回值
  - 返回值是局部变量
  - 具名返回值就是当前函数可以利用的 局部标识符、局部变量，外边的函数，如main函数不可使用

```txt
    return
```

  - 一般来说，是函数能够执行的最后一条语句，其后不在执行语句
    - defer除外
  - 函数执行
    - 要么所有语句执行完成，结束
      - 如果没有return，相当于在最后隐含一个 return
    - 要么碰到 return了，结束
    - 要么函数体代码异常了，非正常结束，此时可能会引起程序崩溃

  - 可以在函数定义中写好返回值参数列表
    - 可以没有标识符，只写类型。但是有时候不便于代码阅读，不知道返回参数的含义
    - 可以和形参一样，写标识符和类型来命名返回值变量，相邻类型相同可以合并写
    - 如果返回值参数列表中只有一个返回参数值类型，小括号可以省略
    - 以上 2 种方式不能混用，也就是返回值参数要么都命名，要么都不要命名

  - return是语句

```txt
            return
```

      - 1、无返回值的函数结束
      - 2、有具名返回值的时使用return表示使用具名返回值返回（相当于return 名称）

```txt
            return value                表示返回某种类型的多个值
            return value1, value2, ...  表示返回多个值
```

    - return的返回值个数和类型要与返回值的声明一致

# a++ / a--
- a++ 先用a，用完后，a再+1
- ++a a先+1，加完后，在用a
- a--和--a同理，但是Go只允许用 a++ 和 a--
