# 标识符：

定义一个名称，名称指向一个合法值，标识符就是指代。名称本质上是一个字符串，使用方式

- 建立标识符和值之间的对应关系，在编程的时候使用，方便程序员看懂。因为编译后就没有标识符了，替换成地址了，标识符此时没用

## 要求

- 非数字开头
- 数字、字母、特殊符号_、中文，不要轻易使用中文等非英文符号

## 常量标识符

标识符一旦有了对应的值，不可改变

```go
const a int = 100 // 定义常量，等号左边指定类型，不用类型推导，左边a为int类型
a = 200 // 不可以
const s = "gxedu" // 定义常量，等号左边未指定类型，类型推到，依据右边是字面量字符串常量，s是string
```

不可延迟赋值

Go中数组长度不可变，元素能变

## 变量标识符

一旦建立了对应关系，Go类型有要求要求的语言只可以修改成相同类型的合法值

```go
var a = 100 // a表示100 int类型的值
a = 2.1 // 不可以，Go语言有严格的类型要求，Go不是动态语言，不可以中途改变变量类型
```

可延迟赋值，初始化自动赋零值，前提必须指定类型

### 零值

在变量定义中，给定了类型，没有赋初值

| 类型     | 零值  |
| -------- | ----- |
| int      | 0     |
| float64  | 0     |
| string   | ""    |
| bool     | False |
| 引用类型 | nil   |

短格式（常用） short

```go
a := 100	// 用到类型推断，但只能写在函数中，不能写在函数外（如顶层代码）
```

## Go中字面常量

- untyped constant 无类型字面常量，有缺省类型

- 字面常量是人用的表达方式

- 常量 值不可改变

- 数值常量

```go
100 十进制数值表达法，字面量表达法，int
0x64 十六进制表达法
```

- 好处：不同类型间可以混合运算
