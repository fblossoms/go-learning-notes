# 作用域

- 顶层代码：定义全局标识符，包括全局变量、全局常量，这里的全局指的是包全局
    - 包外部使用全局
        - 要求首字母大写
            - 包外可见
                ```go
                fmt.Println()
                fmt.abc()	// 不可以小写，此时不可见
                ```
            - 包内也可见
                ```go
                本包内abc()	Println()
                ```
- 函数中定义的，函数内容使用的局部变量、局部常量
